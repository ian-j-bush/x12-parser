# X12 Parser
A web service that loads in sample X12 837 EDI claims and parses them into one of three formats 
for viewing.

* Plain text
* A rendered claim image
* A list of dropdowns, broken down by loop


  **Currently under development, not yet functional.**

## Tech Stack
* Java 21
* Vanilla JavaScript/HTML frontend
* JUnit test suite
